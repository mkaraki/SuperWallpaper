Usage

SuperWallpaper saves configuration file (SuperWallpaper.txt) in your document directory. If you want to change default access URL, please do the following steps.

  1. Open SuperWallpaper configuration file in notepad or your like an editor.
  2. Write URL into the first line of a configuration file.
  3. Save with UTF8.
  4. Restart SuperWallpaper.
